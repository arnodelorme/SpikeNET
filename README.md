1 - Install
=====

Unless you have Linux 2.4.20, you will have to compile 
SpikeNET (but the process is very easy compared to 
other softwares)

1.1 Compile SpikeNET

	cd src
	make clean
	make
	cd ..

2 - Run Demo
=====

In the main SpikeNET folder type

	SpikeNET
